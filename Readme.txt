

ACCOUNTING SYSTEM FOR MONITORING OBJECTS IN THE BLUETOOTH AREA / BLUETOOTH APRĖPTIES ZONOJE ESANČIŲ OBJEKTŲ STEBĖJIMO APSKAITOS SISTEMA.

Programa kurta naudojant Python ir MariaDB / The program was created using Python and MariaDB. The project was done on rasberry pi

						"""Failų Aprsašymas/File Description"""
						
						
BLE.Desktop - Piktograma   Paleidžia BLESCAN.sh / Shortuct runs the script.
BLEIrasai.py - Valdo duomenų bazės BLEirenginiai lentelę / Controlls Database "BLEirenginai" table.
BLESCAN.sh - Paleidžia programos pirma langą LoginFormaPGR.py / Runs the first form "LoginForma.py".
DarbuotojuIrasai.py -Valdo duomenų bazės Darbuotojai lentelę / Controlls Database "Darbuotojai" table.
GUI.py - Pagrindinis programos Langas, Skenavimo pašto siuntimo ir kitos pagrindinės funkcijos / User interface and other primary functions.
LoginFormaPGR.py - Prisijungimo Langas. Prisijungus Paleidžia GUI.py / User Login form. When logged the script runs GUI.py
ObjektuIrasai.py - Valdo duomenų bazės Objektai lentelę / Controlls Database "Objektai" table.
UzregistruotiPasiimtaBLE.py - Valdo duomenų bazės UzregistruotiPasiimtaObjekta lentelę / Controlls Database "UzregistruotiPasiimtaObjekta" table.
Bluetooth-icon.png - Bluetooth paveikslelis piktogramai BLE.Desktop / An Icon for the shortcut
